dom-create
===

Description
---

Quickly create DOM elements.

Usage
---

```javascript
var domCreate = require('dom-create');

var img = domCreate('img', {
    src: 'some/path.png',
    width: '200',
    height: '200',
    alt: 'some text'
});
```
